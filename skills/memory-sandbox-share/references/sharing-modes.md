# Sharing Modes

## Minimal Matrix

- `L0/L1` internal retrieval:
  raw allow is possible
- `L2` internal retrieval:
  summary-first
- `L2` external share:
  summary-only or deny
- `L3` any ordinary retrieval:
  sandbox or deny

## Sandbox Job Record

- `job_id`
- `request_id`
- `agent_id`
- `input_privacy_levels`
- `output_mode`
- `raw_output_blocked`
- `status`
- `latency_ms`
